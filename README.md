# HitWireProject

## Project Structure

- `src/`      — All C++ source (.cpp) files
- `include/`  — All C++ header (.hpp) files
- `build/`    — Build artifacts
- `CMakeLists.txt` — Build configuration
- `README.md` — This file

## How to Build

```sh
cd project
mkdir -p build
cd build
cmake ..
make
cd ..
./build/hitwire
```
